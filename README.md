- Internal JavaScript

- Console Methods
- User Information Webpage

- Billing Calculator

### Experiment 3

### Experiment 4

- Scope
- Closures
- Try-Catch
- Palindrome Checker

### Experiment 5
- Arrays
- Objects
- Discount Logic
- Shopping Cart
