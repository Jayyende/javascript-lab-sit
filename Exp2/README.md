# Experiment 2 - Billing Calculator

## Aim

Use `var`, `let`, `const`, template literals, and object destructuring to build a Billing Calculator using JavaScript.

---

## Description

This experiment demonstrates the use of modern JavaScript (ES6) features while developing a simple billing calculator.

The application accepts customer details and product information, calculates the subtotal, GST (18%), and displays the final bill dynamically on the webpage.
